# Namma-Kathey (National Pride)

An Android-based educational application developed using Generative AI to promote awareness about regional heroes, freedom fighters, poets, and social reformers from different districts of Karnataka.

## Project Overview

Namma-Kathey is an interactive storytelling application designed for children to learn about Karnataka’s historical personalities and cultural heritage. The app provides engaging stories, quizzes, and interactive features in both Kannada and English.

This project helps bridge the gap between children’s awareness of global fictional characters and the lack of knowledge about local heroes and historical figures.

## Problem Statement

Children today are more familiar with global superheroes and fictional characters but often have limited knowledge about regional heroes, freedom fighters, poets, and reformers from their own districts.

Namma-Kathey solves this problem by creating an engaging digital platform that teaches children local history through storytelling and gamified learning.

## Features

- Interactive Karnataka district map
- Explore district-wise heroes and legends
- Illustrated storybook with swipe navigation
- Kannada and English language support
- Text-to-Speech story narration
- Hero Quiz section
- Heritage Badge rewards
- Statue Finder feature

## Technologies Used

- Android Studio
- Java / Kotlin
- XML
- Generative AI
- JSON
- Text-to-Speech API
- ViewPager2
- Material Design UI

## App Workflow

1. User opens the application
2. Selects a district from Karnataka map
3. Views heroes associated with that district
4. Reads illustrated stories
5. Listens to stories using Text-to-Speech
6. Attempts quizzes
7. Earns Heritage Badge
